# Web (Vite + React)

Minimal Vite + React frontend for the invoicing app.

Prerequisites

- Node.js and npm

Quick start

```bash
npm install
npm run dev
```

Build

```bash
npm run build
```

Notes

- Entry is `src/main.jsx`; config in `vite.config.js`.
- ESLint setup is included (see `eslint.config.js`).
